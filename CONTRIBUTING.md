# Contributing to CodeForge

Thank you for considering contributing to CodeForge!

## Getting Started

1. Fork the repository

2. Clone your fork

```bash
git clone https://github.com/YOUR_USERNAME/CodeForge.git
```

3. Install dependencies

```bash
npm install
```

4. Run locally

```bash
npm run dev
```

---

## Creating a Feature

Create a new branch:

```bash
git checkout -b feature/your-feature-name
```

Examples:

```
feature/swagger-support
feature/entity-relationships
feature/jwt-generator
feature/nestjs-generator
```

---

## Commit Messages

Examples:

```
feat: add JWT generator
fix: resolve DTO generation issue
refactor: simplify entity generator
docs: update README
```

---

## Pull Requests

- Keep PRs focused.
- Write clear commit messages.
- Test your changes before submitting.
- Update documentation when needed.
